Optimise every upgrade before you spend a single yen

The Anime Vanguards investment optimizer is in development. The tool will turn damage, survivability, cost-efficiency, and synergy data into a ranked upgrade plan so you always know which unit deserves resources next.

Development roadmap

  • • Algorithm prototype completed; currently tuning weights and payoff curves.
  • • Interface design in progress with drag-and-drop lists and preset weight templates.
  • • Closed beta planned for the next content update—join Discord @av-helper to get an invite.

Problems we are solving

  • • Too many upgrade candidates and not enough yen to rank them manually.
  • • Hard to compare long-term payoff when damage, utility, and survivability all matter.
  • • Players want a bridge between the recommender output, team analysis, and actual resource spend.

Planned feature set

  • • Weighted scoring to balance raw DPS, sustain, cost efficiency, and beginner friendliness.
  • • Import teams from the recommender or synergy analyzer and instantly rank upgrade priority.
  • • Save multiple investment strategies (e.g. “Story push”, “End-game raid”) for quick reuse.
