# 角色图片资源规范

## 📁 文件命名规则

### 基本规则
- **小写字母**：所有文件名必须小写
- **连字符分隔**：多个单词用连字符 `-` 连接
- **无空格**：不能包含空格
- **PNG格式**：推荐使用透明背景的PNG格式

### 命名示例

| 角色显示名 | 文件名 |
|---------|--------|
| Gojo | `gojo.png` |
| Sukuna | `sukuna.png` |
| Song Jinwu | `song-jinwu.png` |
| Igros | `igros.png` |
| Alucard | `alucard.png` |
| Cha-In | `cha-in.png` |
| Sprintwagon | `sprintwagon.png` |
| Alligator | `alligator.png` |

## 🖼️ 图片规格要求

### 角色头像 (characters/)
- **尺寸**: 200x200px（推荐）
- **最小**: 150x150px
- **格式**: PNG（透明背景优先）
- **质量**: 高清，无水印
- **内容**: 角色正面头像或半身像

### 首页大图 (heroes/)
- **homepage-hero.png**: 800x600px - 1200x900px
- **featured-units.png**: 角色合集/拼贴图

### 工具图标 (tools/)
- **尺寸**: SVG（矢量图优先）或 64x64px PNG
- **格式**: SVG / PNG

### 背景装饰 (backgrounds/)
- **尺寸**: 根据需求
- **格式**: PNG / WebP
- **用途**: 渐变叠加、装饰性背景

## 🔍 获取图片来源

1. **官方Wiki**: https://animevanguards.fandom.com/wiki/
2. **游戏内截图**: 直接从游戏中截取
3. **社区资源**: Discord/Reddit
4. **AI生成**: 使用一致性的风格

## ⚠️ 注意事项

- 保持图片风格一致（同一来源优先）
- 确保有使用权限（避免版权问题）
- 定期备份图片资源
- 压缩图片以提高加载速度（推荐使用 TinyPNG）
